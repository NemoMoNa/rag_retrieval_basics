# RAG Retrieval Basics

This project demonstrates a simple Retrieval-Augmented Generation (RAG) pipeline.

## Overview

Pipeline:
1. Document chunking
2. Retrieval (TF-IDF / similarity)
3. Prompt construction
4. LLM-based answer generation

## Key Learning

RAG performance is highly dependent on retrieval quality.

Better retrieval → Better context → Better answers

## Tech Stack

- Python
- scikit-learn (TF-IDF)
- Hugging Face Transformers
- Jupyter Notebook
